# Claude API Proxy для теста женственности

Этот прокси-сервер скрывает твой API-ключ Claude от пользователей сайта.

## Зачем это нужно

API-ключ нельзя класть прямо в HTML на GitHub — его все увидят и украдут. Прокси на Vercel получает запрос от сайта, добавляет ключ из секретных переменных, и отправляет Claude. Ответ возвращается обратно на сайт.

## Что внутри

- `api/analyze.js` — серверная функция, которая обращается к Claude
- `package.json` — настройка для Vercel

## Как развернуть на Vercel — пошагово

### Шаг 1 — Создай новый репозиторий на GitHub

1. Зайди на github.com → создай новый репозиторий, например `kamasutra-ai-proxy`
2. Сделай его **публичным** (для бесплатного Vercel)
3. Загрузи в него файлы: папку `api/` и `package.json`

### Шаг 2 — Подключи репозиторий к Vercel

1. Зайди на vercel.com → войди через GitHub
2. Нажми **"Add New" → "Project"**
3. Найди свой репозиторий `kamasutra-ai-proxy` → нажми **"Import"**
4. Никаких настроек менять не нужно — нажми **"Deploy"**
5. Подожди 1 минуту

### Шаг 3 — Добавь API-ключ в переменные окружения

1. После деплоя открой свой проект на Vercel
2. Нажми **"Settings"** в верхнем меню
3. Слева выбери **"Environment Variables"**
4. Добавь переменную:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** твой API-ключ (начинается с `sk-ant-...`)
   - **Environments:** оставь все три галочки (Production, Preview, Development)
5. Нажми **"Save"**

### Шаг 4 — Передеплой проекта

После добавления переменной нужно сделать redeploy, чтобы она применилась:
1. Зайди во вкладку **"Deployments"**
2. Найди последний деплой → нажми **"⋯" → "Redeploy"**
3. Подтверди

### Шаг 5 — Скопируй URL и вставь в HTML

После redeploy в верху страницы будет адрес твоего проекта, например:
`https://kamasutra-ai-proxy.vercel.app`

Твой API URL будет:
`https://kamasutra-ai-proxy.vercel.app/api/analyze`

Открой `index.html` на GitHub Pages в репозитории `kamasutra-test`, найди строку:
```js
const CLAUDE_API_URL = '';
```
И вставь свой URL:
```js
const CLAUDE_API_URL = 'https://kamasutra-ai-proxy.vercel.app/api/analyze';
```

Сохрани (commit) — и через минуту сайт начнёт использовать Claude API.

## Проверка что работает

Открой свой сайт kamasutra-test, пройди тест. На экране результата должна появиться надпись "Эльвира готовит твою персональную расшифровку…" с тремя пульсирующими точками. Через 3-5 секунд текст заменится на персональный.

Если ничего не меняется — открой консоль (F12) и посмотри ошибки.

10 000 пользователей = $10
